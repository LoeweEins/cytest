================================================================================
end2end 自动化用例总表（与代码同步说明）
================================================================================

一、文档位置与格式
--------------------------------------------------------------------------------
1. Excel 可直接打开：同目录下 END2END_TEST_CASES.csv（UTF-8 BOM，中文列名）。
2. 本文件为纯文本总览，便于 diff 与快速检索。

二、被测对象与运行方式
--------------------------------------------------------------------------------
- 默认对接 Saleor GraphQL（远程 demo 或自建），详见 ENV_SETUP.md。
- 单套件示例：
    python3 -m cytest cases/end2end/10_auth
    python3 -m cytest cases/end2end/60_web_ui
- UI 套件依赖：selenium、本机 Chrome、匹配版本的 chromedriver（可用环境变量
  CHROMEDRIVER_PATH / CHROME_HEADLESS 等，见 lib/web_ui.py 与 60_web_ui/__st__.py）。

三、用例清单（26 条）
--------------------------------------------------------------------------------

【10_auth — auth_cases.py】
  cAUTH0001  AUTH-0001-匿名查询 Shop 基本信息
             标签: API,冒烟 | 验证匿名 shop 查询与 shop.name
  cAUTH0002  AUTH-0002-tokenCreate
             标签: API,回归 | 需测试账号；生成 JWT
  cAUTH0003  AUTH-0003-tokenVerify
             标签: API,回归 | 需 access token
  cAUTH0004  AUTH-0004-tokenRefresh
             标签: API,回归,高风险 | 需 refresh；刷新后 verify
  cAUTH0005  AUTH-0005-me（Bearer）
             标签: API,回归 | 需 access token
